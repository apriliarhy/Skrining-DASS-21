Kuisioner DASS-21

Petunjuk: Jawablah setiap pernyataan berdasarkan kondisi Anda selama seminggu terakhir.

Skor:
0 = Tidak pernah    1 = Kadang-kadang    2 = Cukup sering    3 = Sangat sering atau hampir selalu

1. Saya merasa sulit untuk bersantai.

2. Saya merasa mulut saya kering.

3. Saya tidak dapat mengalami hal-hal yang menyenangkan seperti biasanya.

4. Saya merasa sulit untuk bernapas tanpa alasan yang jelas.

5. Saya merasa tidak berharga.

6. Saya merasa cemas tanpa alasan yang jelas.

7. Saya merasakan jantung berdebar tanpa alasan.

8. Saya merasa mudah tersinggung.

9. Saya merasa panik tanpa sebab.

10. Saya merasa tidak ada harapan sama sekali.

11. Saya merasa reaksi saya berlebihan terhadap situasi.

12. Saya merasa sulit untuk tenang setelah sesuatu mengganggu saya.

13. Saya merasa hidup tidak berarti.

14. Saya merasa saya menggunakan banyak energi untuk mengatasi kekhawatiran.

15. Saya gemetar (misalnya, tangan).

16. Saya merasa sedih dan murung.

17. Saya merasa tidak ada yang bisa saya nantikan.

18. Saya merasa mudah terganggu.

19. Saya merasa takut tanpa alasan yang jelas.

20. Saya merasa saya akan panik dan membuat malu diri saya sendiri.

21. Saya merasa tidak bersemangat untuk melakukan apa pun.
